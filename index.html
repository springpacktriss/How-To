How To Code

This is my second website, where I will put different "how to" subjects on here.

Note: for this first "how to" subject, I can not simply put the code on the page, or the computer will think that it is part of the HTML coding that you can not see unless you inspect the page. So, if there is an asterick (*), then you do not need the dots that are in the code that I am showing to you.

For coding you need...

  • An html tag, like this: *<.html> <./html>

  • A head tag, like this: *<.head> <./head>

  • A body tag, like this: *<.body> <./body>

  • A title tag, like this: *<.title> <./title>

  • Different size tags, like these: *<.h1> <./h1>, *<.h2> <./h2>, etc. Note: this only goes up to six, and "h1" is the biggest, and "h6" is the smallest.

  • list tags, like the ones that I am using like this: *<.ul> <./ul>, *<.ol> <./ol>, *<.li> <./li>

There are many more tags, but these are the ones that you need to know to get started.

If you want to learn more of the details, then go here. They are only the first ten of the eighty-some videos that are on Myepassignments.com, a completely free homeschooling site.

Now, you can put these tags together like this...

Remember, don't type the dots in the tags!

<.html>

<.head>

<.title>My shopping list<./title>

<./head>

<.body>

<.h1>My shopping list<./h1>

<.ul>

<.li>item 1<./li>

<.li>item 2<./li>

<.li>item 3<./li>

<./ul>

<./html>

...And it'll look like this! (It just won't have any color except for black.)
